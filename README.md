# Web-Insight-Scraper 🔍
**Turn Raw Web Data into Actionable AI Insights**
Web-Insight-Scraper is a high-performance Python tool designed for data engineers and market analysts. It bridges the gap between raw web scraping and strategic decision-making by using the **Gemini 2.5 Flash API** to automatically categorize data and extract market trends.
## 🚀 Value Proposition
Manual research takes hours. This tool automates the "Fetch -> Clean -> Analyze" pipeline:
 * **Instant Extraction:** Pulls headings, product titles, and links from any URL.
 * **AI-Powered Intelligence:** Categorizes raw data into industry segments.
 * **Trend Spotting:** Identifies the "Most Important Trends" using Google's latest LLMs.
 * **Ready-to-Use Output:** Generates structured CSVs for data pipelines and Markdown reports for stakeholders.
## 🛠️ Installation
 1. **Clone the repository:**
   ```bash
   git clone [https://github.com/hammadvirk-dev/Web-Insight-Scraper.git](https://github.com/hammadvirk-dev/Web-Insight-Scraper.git)
   cd Web-Insight-Scraper
   
   ```
 2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   
   ```
 3. **Set up your API Key:**
   Ensure your environment has access to a Gemini API Key.
 4. **Run the tool:**
   ```bash
   python scraper_ai.py
   
   ```
## 📊 How it Works
 1. **Scraper:** Uses requests and BeautifulSoup to parse HTML.
 2. **Processor:** Pandas structures the data for export.
 3. **Inference:** Google Generative AI processes the top nodes to find patterns.
